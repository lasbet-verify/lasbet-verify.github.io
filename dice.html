Using the formula hmac_sha256(server_seed, client_seed:nonce),

  1. Take the first 8 characters, resulting in {{ baseHash }}.
  2. Convert this to decimal using number = int(hash[0:8], 16), which gives {{ baseNumber }}.
  3. Divide by 2^32, resulting in {{ normalizedResult }}.
  4. Multiply by 10000, resulting in {{ scaledResult }}.
  5. Finally, calculate dice = floor(result * 10000) / 100, which gives {{ numResultDisplay }}.
